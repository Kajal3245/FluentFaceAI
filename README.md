# FluentFace AI

An AI-powered English Speaking Tutor built using React, FastAPI, Gemini AI, and Speech Recognition.

## 🚀 Live Demo
https://fluent-face-ai.vercel.app


## Features
- 🎤 Speech Recognition
- 🤖 AI Grammar Correction
- 🗣️ AI Voice Response
- 😊 Animated AI Avatar
- 📈 Grammar Score
- ❓ Follow-up Questions

## Technologies Used
- React (Vite)
- FastAPI
- Python
- Google Gemini API
- Web Speech API
